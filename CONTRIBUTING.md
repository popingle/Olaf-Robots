# Contributing to Olaf Robotics

Thank you for your interest in contributing to Olaf Robotics! We welcome contributions from everyone.

## How to Contribute

1.  **Fork the repository** and create your branch from `master`.
2.  **Install dependencies** including C++ build tools and Python packages.
3.  **Make your changes**. If you're adding a new feature, please include tests.
4.  **Run tests** to ensure everything is working as expected.
5.  **Submit a pull request** with a clear description of your changes.

## Technical Standards

*   **C++**: Follow the C++17 standard. Use `clang-format` for code formatting.
*   **Python**: Follow PEP 8. Use `black` for formatting and `mypy` for type checking.
*   **Documentation**: Update the README and any relevant documentation in the `docs/` folder.

## Code of Conduct

Please be respectful and professional in all interactions within the project.
